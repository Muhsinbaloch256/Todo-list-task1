# Todo List App (Python)

A simple command-line Todo List application built using Python.

## Features
- Add new tasks
- View all added tasks with numbering
- Exit the application safely
- Handles empty task list case

## How It Works
- Uses a Python list to store tasks
- Runs in a continuous loop until user chooses to exit
- Uses conditional statements to handle user choices

## Technologies Used
- Python (basic concepts: loops, lists, conditionals, input/output)
